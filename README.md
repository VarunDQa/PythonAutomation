# 🧪 Python UI Test Automation

A collection of UI test automation scripts built with Python and Playwright 
as part of hands-on QA automation learning and practice.

---

## 🛠️ Tech Stack
- Python 3.x
- Playwright (Python)
- Pytest
- Selenium (early scripts)

---

## 📁 Test Suites

| File | Application | Coverage |
|------|------------|----------|
| `test_bank_app.py` | Demo Banking App | Login, transactions, account management |
| `test_demoblaze.py` | Demoblaze E-commerce | Product selection, cart, checkout |
| `test_saucedemo.py` | Sauce Demo | Login, inventory, sorting |
| `test_web_automation3.py` | Web Automation Practice | Form interactions, navigation |

---

## 🚀 How to Run

```bash
pip install playwright pytest pytest-playwright
playwright install
pytest -v
```

---

## 📌 Note
These scripts represent practical learning exercises across multiple 
demo applications. For a structured framework with POM architecture, 
see [playwright-automation-framework](https://github.com/VarunDQa/playwright-automation-framework).
